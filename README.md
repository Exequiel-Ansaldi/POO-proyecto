## 🚀 Featured Projects

### 🍽 Restaurant Management System (OOP)

A complete object-oriented system developed in Java to manage a restaurant.

**Main Features:**
- Table management
- Reservation system
- Schedule organization
- Event planning
- User roles (admin, staff, clients)
- Business logic validation

**Concepts Applied:**
- Inheritance and polymorphism
- Abstract classes
- Encapsulation
- Collections (ArrayList, HashMap)
- Association between classes
- UML-based design implementation

This project simulates a real-world restaurant management scenario and was designed following object-oriented principles.
